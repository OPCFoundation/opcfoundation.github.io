About the OPC Foundation and OPC Technology

Welcome to the OPC Foundation GitHub Pages.

The OPC Foundation is a non-profit technology standards organization responsible for OPC Unified Architecture and the Classic COM-based specifications DA, HDA, A&E, Batch, Security, XMLDA, and others.

Unified Architecture (UA)

UA is the latest technology for securely sharing data between different applications, systems, or platforms. Data can be simple scalar types, structures, or complex objects. Interoperability is assured via the open architecture, shared stacks, and a Certification program.

The OPC UA Type Repository can be found https://opcfoundation.github.io/UA-TypeRepository/.
